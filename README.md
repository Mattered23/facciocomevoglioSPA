Repository per progetti Marino-Odierna

Descrizione:

Questa repository contiene vari circuiti sviluppati tramite Arduino, ognuno con compiti diversi

Obiettivo:

Imparare ad utilizzare Tinkercad e i suoi componenti + scrivere codice con Arduino

Componenti utilizzati:

Arduino Uno 
Breadboard
Resistori
Cavi
Potenziometro

Come usare il progetto:

1.Copiare i link all'interno della repository su github
2.Incollare i link sul browser
3.Aprire il progetto sul proprio computer tramite il pulsante 
4. Avviare la simulazione
